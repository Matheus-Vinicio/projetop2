CÓDIGO NO TEXTO

<code>
Essa tag é utilizada para colocarmos um código visível ao leitor, porém, ela não é bem estruturada ou não se preocupa com o visual, por isso utilizamos também a tag <pre>

CITAÇÕES

CITAÇÃO SIMPLES

Para citar a frase de alguém no texto, utilizamos a tag <q>, por exemplo: Eu sou o caminho, a verdade e a vida!

CITAÇÃO ELABORADA

Uma citação elaborada usamos a tag <blockquote> já que ela desloca o texto dando uma maior enfase a frase citada. Por exemplo:

Eu sou o caminho a verdade e a vida
Ou seja, tem um caráter box-level

ABREVIAÇÕES

Quando passamos o mouse em cima e aparece uma mensagem oculta? sabe como fazer isso? É simples, basta utilizar a tag <abbr>
Por exemplo: PASSE O MOUSE AQUI

TEXTO INVERTIDO

Para deixarmos as pessoas um pouco confusas e inverter o texto, podemos utilizar a tag <bdo dir="rtl">

LISTAS

Utilizamos listas para tudo nesta vida, portanto, saber como colocá-las em HTML5 é de extrema importância

Listas Ordenadas

Utilizamos a tag <ol> para as listas ordenadas e <li> para o itens

Listas Não-Ordenadas

Utilizamos a tag <ul> para as listas não-ordenadas e <li> para o itens

Listas de Definições

Utilizamos a tag <dl> a tag <dt> para os itens e a tag <dd> para a definição

CARACTERES ESPECIAIS

Para colocar caracteres especiais no site, não podemos utilizar eles na linha de código, mas devemos utilizar certos "comandos" para conseguir torná-los visíveis para os visitantes.

LINKS

ÂNCORAS

Para nos referimos a um link ou emplementarmos um link à um determinado elemento, utilizado a tag <a href="">

Especificações

Para uma PÁGINA INTERNA, utlizamos a tag dentro do href rel="prev/next"

Para destinarmos se a página será aberta em outra aba ou na própria aba, utilizamos dentro do href também tag target="_blank/_self..."

Para um página que não temos confiança, utilizamos a relação rel="nofollow"
