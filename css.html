TIPOS DE CSS

CSS INLINE

O CSS INLINE é o mais báscio que encontramos nos códigos, vemos ele normalmente após a utilização de uma tag, por exemplo:
<p "style="font-size:20px;" ">

Geralmente, ele é o predominante quando há outros estilos de CSS envolvidos.

CSS INTERNO

Normalmente, é um CSS mais rebuscado, porém, não tanto utilizado, fica na página principal com o HTMl5

No grupo <head>, adicionamos a tag <style> e iniciamos as tag do CSS.

                        
                            <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>CSS3</title>
                            
                                <STYLE>
                                    CSS INTENRO
                                </STYLE>
                            
                            </head>

CSS EXTERNO

É o estilo de CSS mais utilizado em todo o mundo, é um comando que puxa o estilo de outro arquivo:

<link rel="stylesheet" href="">

                            
                            <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>CSS3</title>
                            <link rel="stylesheet" href="style2.css">
                            </head>

Ao criar a nova Página .css, devemos colocar o seguinte comando:

@charset "UTF-8"
